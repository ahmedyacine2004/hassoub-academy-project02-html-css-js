# YouTube Frontend Template

A super simple YouTube-style frontend built with HTML and Bootstrap.

## Preview

Add your screenshots here.

## Features

* YouTube-like homepage layout
* Video cards with thumbnails
* Navbar with logo and search bar
* Video page layout
* Responsive on all devices

## How to Use

1. Clone the project.
2. Open `index.html`.
3. Replace images and text with your own.

## Folders

* `index.html` main page
* `assets/` for images
* `css/` for styles

## Notes

This is a frontend template only. No backend, no API.
